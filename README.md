Mở file `index.html` bằng trình duyệt để xem.

## Nội dung

- 01 · Tổng quan về Fanpage
- 02 · Hạn chế & Giải pháp
- 03 · Đề xuất khác
- 04 · Review chi tiết
- 05 · Content Plan
- 06 · Content Angle
